# Movie Explorer App 

A simple and clean React application to explore your favorite movies using the [OMDb API](http://www.omdbapi.com/). Search for movies and view their poster and release year.  

![Movie Explorer Screenshot](screenshot.png)  

---

## Features

- Search for movies by title.
- View movie thumbnail (poster) and release year.
- Responsive and minimal UI built with React.
- Powered by [OMDb API](http://www.omdbapi.com/).
